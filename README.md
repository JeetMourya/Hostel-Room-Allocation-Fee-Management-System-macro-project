## 🎯 Overview

The **Hostel Room Allocation & Fee Management System** is a Python-based application designed to simplify hostel administration.

It helps manage:

* Student registrations
* Room allocations
* Fee tracking and records

This system is ideal for colleges and institutions to automate hostel operations efficiently.

---

## ✨ Features

### 🏠 Hostel Management

* Automatic room allocation
* Separate handling for freshers and seniors
* Organized room distribution

### 👨‍🎓 Student Management

* Store and manage student details
* CSV-based data import
* Easy record updates

### 💰 Fee Management

* Track student fee status
* Maintain payment records
* Simple and efficient handling

## 🚀 Quick Start

### Requirements

* Python 3.8+

---

### Setup

```bash
# Clone repository
git clone <your-repo-url>
cd Hostel-Room-Allocation-Fee-Management-System-macro-project

# Create virtual environment
python -m venv .venv

# Activate (Windows)
.venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

---

### Setup Database

```bash
python seed_students.py
```

OR

```bash
python add_test_data.py
```

---

### Run Application

```bash
python hostel_app.py
```

OR

```bash
python start_simple.py
```
